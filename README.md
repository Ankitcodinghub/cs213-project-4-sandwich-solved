Project Description

You will be using JavaFX to develop a software system for ordering sandwiches. For simplicity, let‚Äôs abstract away the details of the customer who is ordering the sandwiches, and abstract away the checkout functionality. However, your software must meet the following functional requirements. -2 points for each requirement not implemented.

<ol>
<li>1) &nbsp;The system shall provide the options of 3 types of sandwiches, Chicken, Beef and Fish. Each type of sandwiches has its basic ingredients. Customers can customize the sandwiches by adding extra ingredients to the sandwiches.</li>
<li>2) &nbsp;The default sandwiches selected on the GUI shall be set to Chicken.</li>
<li>3) &nbsp;Upon the selection of the sandwich type, an image of the sandwich shall be displayed on the GUI, together
with the basic ingredients and the price of the sandwich.
</li>
<li>4) &nbsp;The system shall provide a list of at least 10 extra ingredients for the customer to choose from. The customer
can add or remove the extra ingredients.
</li>
<li>5) &nbsp;Maximum number of extra ingredients added to the selected sandwich will be 6. The customer can also add
the sandwich to the order as is, without adding any extra ingredients.
</li>
<li>6) &nbsp;When the customer is adding the extra ingredients, the system shall not allow the same ingredients to be
added more than once.
</li>
<li>7) &nbsp;The system shall keep track of the sandwich price and display the price while the customer is adding and
removing the ingredients.
</li>
<li>8) &nbsp;The customer can add multiple sandwiches to an order. Each sandwich is identified by a serial number in the
order; that is, each added sandwich is an order line on the order.
</li>
<li>9) &nbsp;The system shall be able to show the order details with a list of sandwiches added in a new window.</li>
<li>10) &nbsp;For each sandwich in the order, the system shall print out the serial number (line number), sandwich type,
the list of basic ingredients, extra ingredients and the price. At the end of the list, a total amount of the order

shall be displayed.
</li>
<li>11) &nbsp;In the order details window, the customer can
<ul>
<li>select a sandwich on the order and add the same sandwich to the order with a new serial number.</li>
<li>select a sandwich on the order and remove the sandwich from the order and reorder the serial numbers on
the order (move everyone up.)
</li>
<li>clear the order; that is, remove all the sandwiches on the order and start a new order; in this case, the
serial number will be reset.

12) The system shall be able to save the order to a file, one order at a time.
</li>
</ul>
</li>
</ol>
